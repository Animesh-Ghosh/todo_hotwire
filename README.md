# todo_hotwire

Single-Page Todo App with Ruby on Rails and Hotwire, built to practice Hotwire.

I am running Ruby 3.2.2, rest can be installed via bundler.

## Installation

1. Clone the repository
2. Run `bin/setup`
3. Run the server

```bash
bin/setup
bin/rails s
```

## Running Tests

```bash
bundle exec rspec
```
